## 🔗 Integração com Jarvis

1. Preencha formulário no dashboard
2. Clique "PROCESSAR"
3. Copie comando que aparece
4. Cole aqui no chat: `Jarvis, processar REF-XXX [dados]`
5. Eu retorno copy aprovada em < 5min
6. Paste resultado de volta
7. Dashboard atualiza ✅
